# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/43fcf85d-0bd7-486a-8e70-6e829c10f97c

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## 🌆 Sobre el Proyecto

**Afterlife Bar** es la recreación web del bar más icónico de *Night City* — ubicado en el **Distrito de Heywood, The Glen**, directamente del universo de **Cyberpunk 2077**.

Un espacio donde los mercenarios beben, los fijers hacen negocios y la música nunca para. Esta web captura esa atmósfera cyberpunk con una experiencia completa: carta de tragos con filtros interactivos, sistema de reservas al estilo terminal, agenda de eventos y visualizador de música animado — todo con estética neón, glitch y CRT.

> *"Nadie llega al Afterlife a tomar agua."*

## ✨ Características

- 🍹 **Carta interactiva** con filtros por categoría (cócteles, shots, sin alcohol)
- 💻 **Formulario de reservas** estilo terminal con validación en tiempo real
- 🎟️ **Sección de eventos** con agenda animada de shows y noches temáticas
- 🎵 **Visualizador de música animado** con barras reactivas al ritmo
- 🍔 **Menú hamburguesa responsive** para navegación móvil fluida
- 👾 **Efectos de glitch y scanlines** sobre la interfaz completa
- 💨 **Partículas de humo en el hero** con animación procedural en canvas
